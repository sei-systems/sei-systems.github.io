## SOC2 & Security Alignment
This project is built following core Trust Services Criteria (TSC) for Security and Availability:
- **Change Management:** All modifications are tracked via Git commits, providing a full audit trail of site evolution.
- **Access Control:** Repository access is restricted and protected by Multi-Factor Authentication (MFA).
- **Transparency:** A dedicated [Compliance Hub](privacy.html) is maintained to outline data handling and AI ethics.
